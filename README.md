# py_papago_img

## Objective
* To translate an image file using Papago Image Translator API

## Cost
* Expected 120 KRW per call

## Limits
| Item | Limit |
|:---|:---|
| formats | jpg, jpeg, png, tiff |
| Maximum image size | 20 MB |
| Maximum image width | 1960 px |
| Maximum image height | 1960 px |
